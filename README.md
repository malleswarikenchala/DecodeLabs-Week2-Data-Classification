# Data Classification Using AI

## Objective
Build a machine learning classification model using supervised learning.

## Tools Used
- Python
- Scikit-learn
- Pandas

## Dataset
Iris Dataset

## Algorithm
Decision Tree Classifier

## Steps Performed
1. Loaded the Iris dataset
2. Split data into training and testing sets
3. Trained a Decision Tree model
4. Made predictions on test data
5. Evaluated model accuracy

## Result
Accuracy: 1.0
